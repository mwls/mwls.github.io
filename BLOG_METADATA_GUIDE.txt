Kashmir Teaching Blog - Adding Metadata to New Posts
=====================================================

When you write a new blog post, add these data attributes to the <section> tag:

<section id="other" class="wrapper style2" data-blog-week="WEEK" data-blog-topics="TOPICS" data-blog-order="ORDER">

Available values:

data-blog-week: 
  - week1  (Days 1-7)
  - week2  (Days 8-14)
  - week3  (Final days, Day 15+)

data-blog-topics: (space-separated list, e.g. "teaching culture")
  - teaching
  - culture
  - tourism
  - social
  - sports
  - travel

data-blog-order:
  - Use a number (1, 2, 3, ..., 18)
  - Use 20+ for epilogue (e.g., 20)
  - Can use decimals for special entries (e.g., 16.5)

Example:
<section id="other" class="wrapper style2" data-blog-week="week2" data-blog-topics="teaching social" data-blog-order="12">

The blog system will:
- Automatically tag the post with week and topic pills
- Allow filtering by time period and topic
- Paginate (3 posts at a time by default)
- Sort newest-first
