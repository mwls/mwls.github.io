Publications

I've been fortunate that I've had significant roles in a few large surveys, for example being the data lead for H-ATLAS or JINGLE and so have worked on many publications. Below I give some basic stats and a list of all the publications I have authored below. All the publication information is taken from my ADS library (Here). You can also find information on these publications on Google Scholar and Cardiff ORCA.
